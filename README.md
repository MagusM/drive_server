# Drive. server
* Nest JS
* ApolloJS
* TypeORM
* GraphQL
* TypeScript
* pnpm

<br />

## Description

This repo will mainly serve the Drive - UI.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
